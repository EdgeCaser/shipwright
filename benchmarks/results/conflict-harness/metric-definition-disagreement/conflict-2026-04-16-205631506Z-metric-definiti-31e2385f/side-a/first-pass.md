# North Star Metric Recommendation

## Decision Frame
Leadership should not force a single undifferentiated "active user" number from the current three incompatible definitions. The recommended north star is a single canonical active-user metric owned cross-functionally, but it should be adopted only through a formal metric-definition decision that explicitly retires the competing definitions for north-star use while preserving them as supporting operational views where needed.

This is the most decision-useful path because the current state makes the same business appear growing, flat, or declining depending on definition. A leadership number that changes meaning by function is not a north star; it is a source of governance failure.

Recommended decision:
1. Establish one canonical active-user definition as the company north star.
2. Freeze that definition for leadership reporting once approved.
3. Reclassify finance, product, and growth variants as diagnostic sub-metrics rather than alternate company truths.
4. Require every executive dashboard and narrative to label the canonical metric separately from function-specific views.

## Unknowns & Evidence Gaps
The case does not provide the actual three definitions, their intended use cases, or which one best reflects enduring customer value. That prevents a fully grounded recommendation for which exact definition should win.

Key unknowns:
- What user action each definition treats as "active"
- Whether any definition is optimized for accounting, product engagement, or acquisition efficiency rather than durable product value
- How often each definition is already used in executive decisions
- Whether the 40% spread is caused by scope, timing, user identity rules, or event thresholds
- Which definition is most stable against tactical reporting changes

Because those gaps remain open, the recommendation is to decide the governance model now and the exact metric definition immediately after a short reconciliation exercise, rather than pretending the current numbers can be cleanly collapsed.

## Pass/Fail Readiness
Pass:
- There is agreement that the company needs one canonical leadership metric
- Each existing definition can be documented and compared on scope, intent, and decision impact
- The selected north-star definition can be applied consistently across reporting periods
- Function-specific definitions are allowed to remain only as secondary diagnostic cuts

Fail:
- Leadership insists on a single number without resolving definition differences
- Teams continue presenting different active-user counts as interchangeable
- The selected metric lacks explicit ownership, written rules, or change control
- Supporting teams are allowed to redefine the north star ad hoc for local goals

## Recommended Next Artifact
Produce a metric decision memo with four parts:
1. A side-by-side definition table for the finance, product, and growth versions of active user
2. A recommendation for the canonical north-star definition and why it is the best company-level measure
3. A governance policy covering ownership, versioning, exceptions, and reporting labels
4. A metric tree showing the canonical north star plus the supporting functional diagnostics

That memo should be the approval object for leadership, after which dashboards and reviews can be updated to use one company-level definition consistently.
