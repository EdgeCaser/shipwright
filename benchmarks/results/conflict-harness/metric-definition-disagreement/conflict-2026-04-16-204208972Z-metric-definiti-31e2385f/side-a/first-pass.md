# North Star Metric Recommendation

## Decision Frame
Leadership needs a single number, but the current situation does not support choosing one of the existing definitions as the company-wide north star today. The three definitions of "active user" are materially incompatible, and the resulting outcomes differ enough to flip the narrative between growth, flat performance, and decline. A single number presented without resolving that semantic conflict would create false precision and make downstream decisions less reliable.

Recommendation: adopt one canonical north star definition through an explicit metric-governance decision, and until that decision is complete, report the three current definitions together with a clearly labeled temporary executive view rather than forcing one disputed number into the north star role.

The best near-term standard is the definition that is most tightly connected to actual product value creation and can be applied consistently across finance, product, and growth. The case packet does not provide enough evidence to determine which existing definition meets that standard, so the immediate decision should be about governance and selection criteria, not about arbitrarily picking a winner.

## Unknowns & Evidence Gaps
The case packet does not specify what each current definition counts, what user behavior each one is intended to represent, or why each function adopted it. Without that, there is no grounded basis for deciding which definition best reflects durable product value.

Key missing evidence:
- The exact logic behind each current definition of "active user"
- Which definition most closely tracks meaningful customer value or product success
- Whether any definition is easier to audit, explain, and reproduce consistently
- Whether one definition is optimized for a functional use case rather than executive decision-making
- How historical reporting, targets, and incentives would be affected by a change

## Pass/Fail Readiness
Pass:
- A cross-functional owner is assigned to metric governance
- The company agrees on selection criteria before choosing the canonical definition
- The chosen definition is documented with clear inclusion and exclusion rules
- Historical restatement and reporting implications are understood before rollout

Fail:
- Leadership insists on a single number before resolving semantic differences
- One function's definition is elevated without an explicit rationale tied to product value
- The metric is adopted without documentation, ownership, or change-control rules
- Teams continue using local definitions while presenting one executive number as if alignment exists

## Recommended Next Artifact
Create a metric decision memo with four parts:
1. The three current definitions written in precise operational terms
2. A comparison against selection criteria such as value alignment, consistency, explainability, and governance fit
3. A recommendation for the canonical north star definition and any secondary diagnostic metrics
4. A transition plan covering historical restatement, reporting changes, and executive communication

That memo should be the decision vehicle. The output should be one approved canonical definition, plus a small metric hierarchy where the north star is distinct from finance, growth, or product diagnostic cuts.
