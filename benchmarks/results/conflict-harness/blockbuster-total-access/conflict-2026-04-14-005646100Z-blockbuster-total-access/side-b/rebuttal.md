# Critique: finding-2

- Target Side: side_a
- Target Claim IDs: side_a-claim-2
- Attack Type: evidence_gap
- Severity: high

## Claim Under Attack
The program can be restructured through tiered return caps, franchisee revenue-sharing, and a streaming pilot to reduce burn from ~$400M to ~$260M/year, extending runway to 36+ months.

## Evidence Or Reason
The claimed burn reduction and runway extension are modeled as if they are proven, but the visible evidence only supports the current ~$400M/year burn, $1.1B debt load, and $85M Q2 2007 loss. The artifact invents savings estimates for return caps, franchisee payments, store closures, and studio renegotiations without evidence on return frequency distribution, subscriber churn under caps, lease termination costs, franchisee acceptance, or studio willingness to subsidize the pivot. Because the recommendation depends on this restructuring being financeable, unsupported arithmetic weakens the core strategic conclusion.
